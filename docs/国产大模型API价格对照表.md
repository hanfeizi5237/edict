## 备注

1. **价格时效性**: 以上价格为整理时参考价格，实际价格以各公司官网实时定价为准
2. **币种说明**: 国内价格为人民币(CNY/¥)，海外价格为美元(USD/$)
3. **汇率参考**: 按 1 USD ≈ 7.2 CNY 计算
